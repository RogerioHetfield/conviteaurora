👑 Convite Digital - Princesa Aurora
Este é um projeto de Convite Digital Interativo desenvolvido para o primeiro aninho da Aurora. O convite conta com elementos visuais temáticos, contagem regressiva em tempo real e botões interativos para facilitar a experiência dos convidados.

🚀 Funcionalidades
Header com Vídeo: Exibição do vídeo personalizado aurora.mp4 no topo do convite.

Contagem Regressiva: Script dinâmico que mostra quantos dias, horas e minutos faltam para o grande dia (07/06/2026).

Botões Interativos:

📍 Localização: Abre o Google Maps diretamente na Chácara do Pedro Taxi.

👤 Confirmar Presença: Link direto para o WhatsApp com mensagem pré-definida.

🎁 Sugestão de Presente: Espaço para lista de presentes ou informações de tamanho.

Design Responsivo: Otimizado para visualização perfeita em dispositivos móveis (WhatsApp/Telegram).

Trilha Sonora: Reprodução de música de fundo que inicia com a interação do usuário.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação de conteúdo e tags de mídia (video/audio).

CSS3: Estilização personalizada com Flexbox, Grid e animações de fadinhas.

JavaScript: Lógica da contagem regressiva e controle de reprodução de mídia.

📂 Estrutura de Arquivos
Plaintext

/
├── index.html          # Código principal do convite
├── aurora.mp4          # Vídeo de destaque (comprimido para ~2.97MB)
├── presentes.html      # Código que vai pra tela dos presentes

🔧 Como usar

Clone este repositório ou baixe os arquivos.

Certifique-se de que o vídeo aurora.mp4 e as imagens PNG estão na mesma pasta que o index.html.

Abra o arquivo index.html no seu navegador para testar.

Para publicar, utilize plataformas como Vercel ou Netlify para gerar o link de compartilhamento.
